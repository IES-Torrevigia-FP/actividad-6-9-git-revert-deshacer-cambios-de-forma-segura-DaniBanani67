git revert HEAD crea un commit nuevo que deshace el último commit.

git reset --hard HEAD~1 en cambio borra el ultimo commit del historial, también puede eliminar cambios
Por eso reset puede causar problemas si el proyecto ya se ha compartido con otras personas
